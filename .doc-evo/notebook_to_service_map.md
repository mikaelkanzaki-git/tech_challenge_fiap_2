# Notebook to Service Map

## Keep from the notebook
- dataset loading
- train/test split
- feature encoding for `arrival_mode`
- Random Forest training
- SMOTE usage
- evaluation metrics

## Do not move into the API service
- exploratory analysis
- plots
- experiment comparison tables
- notebook-specific visual output

## Convert into modules
- preprocessing helpers
- model training
- model persistence
- request and response schemas
- prediction service
